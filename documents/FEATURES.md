# Garments ERP - Key Features

This system is packed with features designed to optimize garment manufacturing.

## 1. Comprehensive Order Management
*   **Multi-Level Tracking**: Track orders from Buyer -> Style -> PO -> Color -> Size.
*   **Dynamic Sizing**: Support for various size categories (XXS to XXL) with specific quantity breakdowns.

## 2. Advanced Production Tracking (RFID/QR)
*   **Bundle Tracking**: Unique identification for every bundle of fabric.
*   **Operation-Level Visibility**: Track not just the bundle, but *exactly which operation* (e.g., Sleeve Attach) is currently being performed and by whom.
*   **Real-Time WIP**: Instant calculation of Work-In-Progress inventory on the floor.

## 3. Human Resources & Skill Management
*   **Detailed Profiles**: Store employee data including designation, department, and join date.
*   **Line Assignment**: Map employees dynamically to production lines.
*   **Skill Grading**: Track operator skill levels (Grade A, B, C) to optimize line balancing.

## 4. Industrial Engineering (IE)
*   **Operation Bulletins**: configure detailed operation sequences and SAM (Standard Allowed Minutes) for every style.
*   **Machine Requirements**: Link operations to specific machine types (SNLS, Overlock, etc.).
*   **Performance Analytics**: Compare Standard Time vs. Actual Time.

## 5. Machine & Maintenance
*   **Asset Registry**: Complete database of all factory machinery.
*   **Maintenance History**: Track service records and mechanic responsibilities.
*   **Status Monitoring**: Flag machines as 'WORKING' or 'BREAKDOWN' for immediate attention.

## 6. Reporting & Analytics
*   **Production Reports**: Daily output summaries.
*   **Wage Calculation Support**: Data ready for piece-rate or hourly wage calculations based on tracked work.
*   **Quality Control**: (Planned) Integration for defect tracking per bundle/operator.

## 7. Security & Access Control
*   **Role-Based Access (RBAC)**: Strict permission enforcement based on user Designation.
*   **Data Isolation**: Ensure Managers see everything while Supervisors see only their lines.
*   **Auditability**: (Planned) Log key actions to specific User IDs.
